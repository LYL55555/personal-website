# Music player

The in-app player loads playlists from **NetEase Cloud Music** via `/api/netease/*` (see [`TEMPLATE_SETUP.md`](./TEMPLATE_SETUP.md) for env vars, compliance, and copyright).

**UI:** `src/components/music-player/MusicPlayer/` — controls, lyrics, playlist input, theme-aware styling.

There is **no** local `/public/music` + `/api/music` pipeline in this template fork.
